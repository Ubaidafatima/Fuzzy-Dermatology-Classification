# Four-Valued Fuzzy Machine Learning Framework for Dermatological Disease Classification

## Overview

This repository contains the Python implementation of the proposed Four-Valued Fuzzy Machine Learning Framework (FVFFF) for dermatological disease classification.

The framework integrates:

- Four-valued fuzzy logic
- Clinically interpretable fuzzy predictors
- Rule-based fuzzy inference
- Machine learning classifiers
- Explainable AI (SHAP)
- External clinical validation
